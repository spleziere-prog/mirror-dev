# mirror-dev

Understand any GitHub repository instantly.

Generate beautiful architecture graphs automatically.

## Features

- Repository visualization
- Architecture graph generation
- Import relationship detection
- Beautiful modern UI
- Static HTML export

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Example

MIRROR.dev scans your repository and generates a visual architecture map automatically.

## Vision

Understanding large codebases should be instant.

MIRROR.dev transforms repositories into readable architecture diagrams.
